Grade Details

Final Grade

In order to get a final grade, a student must have at least 5, both to the Laboratory Grade and Practical Exam.
The Final Grade is computed using the following formula:
  • FG = 0.60 * LG + 0.40 * PE
Where:
  • LG = 0.1 * L2 + 0.2 * L3 + 0.3 * L5 + 0.4 * L7.
  • PE - Practical Exam mark.
Practical Exam Details

On the practical exam the studens will have to develop a mobile app based on the requested features using a platform of choise (from: Android, iOS, React-Native).
Exam Requirements:
  • Duration: 120 minutes
  • The students must obey the schedule established, attend the scheduled first day for their group (see here).
    • The only allowed absences/changes are due to medical reasons.
  • Each development machine should have the following software installed:
    • Git - used to retrieve the server source code.
    • Android Studio, Xcode or React-Native - based on development choices.
    • Golang - to be able to execute the server side.
    • Node - to be able to execute the server side (node version at least 8.0.0 and npm at least 4.0.0).
Re-examination
Students that have the Laboratory Grade less then 5:
  • Will take an exam similar to the requirements presented above but will have to implement more/complex features, in the same amount of time.
  • The mark obtained on this exam will represent the Final Grade.
Students that have the Laboratory Grade greater or equal to 5, will have the choice:
  • To take an exam similar to the one from the normal examination session, only to increase the practical mark.
    • The Final Grade will be computed using the same formula presented above.
  • To take the more complex approach:
    • Implement more/complex features in the same amount of time.
    • The mark obtained will represent the Final Grade.
