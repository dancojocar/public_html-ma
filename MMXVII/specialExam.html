Exam - Special Session Details

!!! Note that based on the faculty requirements you need to fill and submit a request, till June 19th, to the secretaries, in order to participate to the special exam session.

The exam requirements are similar to the ones from the normal session.
Requirements:
  • Duration: 120 minutes
  • Each development machine should have the following software installed:
    • Git - used to retrieve the server source code.
    • Android Studio, Xcode or React-Native - based on development choices.
    • Golang - to be able to execute the server side.
    • Node - to be able to execute the server side.
Please ensure, using a server from an exam from the normal session (Eg. https://github.com/dancojocar/exam0221a), that your machine is properly configured and able to execute the nodeJs server.
Also you should know how to connect your mobile device to your development machine (ensure that you are on the same Wifi network and are able to communicate).

Steps to execute the server:
  • Clone the exam repo. Eg. git clone https://github.com/dancojocar/exam0221a
  • Change the directory to server: cd server
  • Update the npm dependencies by running: npm install
  • Execute the server, using: npm start
